# Game-Shelf

This web application allows users to create and manage a list of board games they want to play. The application's user-friendly interface enables users to manage their game list effectively.

## Functionality

### Users

- Users can search for games by name or AKAs, and track the games they would like to play by adding them to their 'Game Shelf'.
- Users can log in and out of their accounts, search for games, add/remove games to their game shelf.
- Users can manage their profile.
  - Change their First/Last name
  - Update their password

### Administrators

- Admin's can manage other user's accounts in the following ways:
  - Disable/Enable
  - Set/Unset Admin
  - Manage any user's profile
